# 安全说明

本项目是静态 Codex Pet 资源包，不包含后端服务、网络接口或运行时代码。主要风险来自素材来源、供应链文件和文档中的误导性说明。

## 报告问题

如果你发现以下问题，请通过 Issue 或私下联系维护者报告：

- 仓库中意外包含密钥、令牌、个人信息或私有素材。
- 文件伪装、异常二进制内容或可疑脚本。
- 授权说明与素材来源存在明显冲突。
- 安装说明可能导致用户覆盖重要文件。

报告时请提供受影响文件、复现步骤和你建议的修复方式。

## 不属于安全漏洞的内容

- 宠物动作帧不好看、帧率不理想或样式不统一。
- README 文案措辞建议。
- 普通的图集校验失败。

这些问题请使用普通 Issue 或 PR 处理。
